Become a platform

Your API has great value. Start exposing it with the more secure and more reliable existing protocol. Become an OAuth2 Provider!

This service will provide an appropriate solution to quickly and securely expose your API under a recommended implementation of the OAuth2 protocol (based on the final specification RFC6749).

Highly customizable

Our solution will integrate nicely in your existing API and data structure. Manage OAuth scopes and permissions easily with middleware. We will provide SDK for main backend language (Node.js, PHP, Ruby, Python, Go etc...)

Exemple with the /me endpoint in Node.js: 

app.get('/me', OAuthProvider.OAuth2.check({
  scope: ['raw_info']
}), function (req, res, next) {
  user = req.session.user;
  if (typeof user !== 'undefined') {
    response = {
      username: user.username,
      avatar: user.image_path,
    }
    if (req.OAuth2.scope.indexOf('email') != -1) {
      response.email = user.email;
    }
    res.status(200).json(response);
  }
  else {
    res.status(403).send('Not authenticated');
  }
});
