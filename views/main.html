Be smart & start providing OAuth now

If you are beginning with a project, start providing OAuth from day one, without the usual hassle. If your API is already mature, our OAuth Server solution will allow you to give API accesses to your partners and developers community, in a very simple way.

Exemple with the /me endpoint in Node.js: 

app.get('/me', OAuthProvider.OAuth2.check({
  scope: ['raw_info']
}), function (req, res, next) {
  user = req.session.user;
  if (typeof user !== 'undefined') {
    response = {
      username: user.username,
      avatar: user.image_path,
    }
    if (req.OAuth2.scope.indexOf('email') != -1) {
      response.email = user.email;
    }
    res.status(200).json(response);
  }
  else {
    res.status(403).send('Not authenticated');
  }
});
